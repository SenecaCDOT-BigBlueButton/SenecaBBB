How To Add Department

Note: Only System Admin and Department Admin have permissions to create new department.

  1. Click on the "Departments" button on the left side menu bar
  2. On "Departments" page, click on the "Add department" button will redirect you to the "Create Department" page
  3. Input valid department code (e.g. ICT) and department name in the form.
  4. Click the "Save" button to save department, click the "Reset" to clear the form, or click "Cancel" button to navigate back to "Departments" page.
