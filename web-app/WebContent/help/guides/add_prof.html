How To Add Professor

Note: Only System Admin and Department Admin have permissions to create professors.

  1. Click on the "Subjects" button on the left side menu bar
  2. On "Subjects" page, you should see a table of all current course sections in system
  3. Click the "Manage Professors" Button, this would bring you to the "Professor List" page which contains all professors in system.
  4. Click the "Add Professor" Button, this would bring you to the "Create Professor" form which allows you to bind a professor's employee id with a subject section.
  5. Type in a valid Seneca employee username and choose a section from the section list.
  6. Click the "Create" button to create professor, click the "Reset" to clear the form, or click "Cancel" button to navigate back to "Professor List" page.
  7. If create successfully, you should be able to find the professor on professor list.
