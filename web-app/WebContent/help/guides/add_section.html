How To Add Section

Note: Only System Admin and Department Admin have permissions to add sections.

  1. Click on the "Subjects" button on the left side menu bar
  2. On "Subjects" page, you should see a table of all current course sections in system
  3. Click the "Add Section" Button, this would bring you to the "Create Section" page which containing lists of available "Course ID","Semester ID", and "Department ID".
  4. Click the "Add Subject" Button, this would bring you to the "Create Subject" form which allows you to create a new subject.
  5. Type in valid section ID (maximum 2 upper case characters, e.g. A, AB) on the "Section ID" input box.
  6. Click the "Create" button to create subject, click the "Reset" to clear the form, or click "Cancel" button to navigate back to "Subject List" page.
  7. If create successfully, you should be able to find the new section on sections list.
