How To Add Students To Class From External File

Note: Only professors have permission to add students and guests to their own classes.

  1. Click on the "Class Settings" button on the left side menu bar
  2. On "Class Settings" page, you should see a message to ask you choose a class to add students.
  3. Click on the class dropdown list to choose a class to add students.
  4. You should see a list of students (if you have already added) in current class.
  5. Click on the "Choose File" button to select external file to load. System only accept CSV comma separated file and XLS tab separated file format!
  6. If file uploaded successfully, you should be able to find all added students on the student list table.
  7. You could also add one student at a time using the "Search Student To Add" search box by typing in valid Seneca student full username into the search box. then press "Enter" key or click the search icon to add student to your class.
  8. An email notification, including professor name and class start time, would be sent to your new added students.
