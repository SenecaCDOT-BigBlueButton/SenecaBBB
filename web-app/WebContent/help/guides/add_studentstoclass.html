How To Add Students To Class

Note: Only professors have permission to add students and guests to their own classes.

  1. Option 1:
    • Click on the "Events" button on the left side menu bar.
    • Find lecture you want to add students using the "Filter entries" function by inputing event title or description on the "Filter entries" box, or you could click on any sortable table column header to make your search easier.
    • Press the details icon on the row you want to edit, this would bring you to the "Current Event" page.
  2. Option 2:
    • Click on the "Calendar" button on the left side menu bar.
    • Find lecture you want to add students, you can check or uncheck the filter options to help your search.
    • Press the event on calendar you want to edit, this would bring you to the "Current Event" page.
  3. Click on the "Manage Student" button would bring you to the "Add Student" page.
  4. On "Add Student" page, you should see a list of students (if you have already added) in your class.
  5. To add new student to your class, fill valid Seneca student full username into the "Add User" input box.
  6. Press "Enter" key or click the search icon to add student to your class.
  7. A "Success", "User Not Found", or "User already in database" information would show on top of the page, if success, you should be be able to find the student on the table.
  8. To add Guest users, please input full or partial username into the second "Search User" input box to search any matched users in system.
  9. If system found any matched guest users, it would display a table of search result. You could click on the green color "Add" icon to add the guest to attendee list
  10. An email notification, including professor name and class start time, would be sent to your student.
