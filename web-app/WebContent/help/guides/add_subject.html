How To Add Subject

Note: Only System Admin and Department Admin have permissions to manage subjects.

  1. Click on the "Subjects" button on the left side menu bar
  2. On "Subjects" page, you should see a table of all current course sections in system
  3. Click the "Manage Subjects" Button, this would bring you to the "Subject List" page which contains all subjects available in system.
  4. Click the "Add Subject" Button, this would bring you to the "Create Subject" form which allows you to create a new subject.
  5. Type in valid subject code (e.g. IPC144) and subject name on the input box.
  6. Click the "Create" button to create subject, click the "Reset" to clear the form, or click "Cancel" button to navigate back to "Subject List" page.
  7. If create successfully, you should be able to find the new subject on subject list.
