How To Add User

Note: You can only add internal users (Seneca students,employees) to the system, if you want to add a guest, please visit Create Guest Account.

  1. Click on the "Manage Users" button on the left side menu bar
  2. Fill valid Seneca student or employee full username into the "Add User" input box.
  3. Press "Enter" key or click the search icon to add user to system
  4. A "Success", "User Not Found", or "User already in database" information would show on top of the page, if success, you should be be able to find the user on the table.
  5. You could also use the "Filter entries" function to find if the new user is added or not.
