How To Create Quick Meeting

Note: By default, quick meeting is designated for current day's short single meeting only. If you want to create scheduled meetings, please visit Create Meeting help page.

  1. Click on the "Quick Meeting" button on the left side menu bar.
  2. Fill valid meeting title in the form.
  3. Click on the "Start Time" box and select start time from dropdown list.
  4. Fill in minutes of meeting duration.
  5. Click on the "Create" button to create quick meeting, you would be redirected to "Add Meeting Attendee" page
