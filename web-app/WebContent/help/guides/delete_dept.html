How To Delete Department

Note: Only System Admin and Department Admin have permissions to remove existing departments.

  1. Click on the "Departments" button on the left side menu bar
  2. On "Departments" page, you should see a table of all current departments in system
  3. Find the department you want to remove using the "Filter entries" function by inputing relevant department code or name on the "Filter entries" box, or you could click on any sortable table column header to make your search easier.
  4. Press the remove icon on the row you want to delete.
  5. After system pops up a confirmation window, click on "Ok" to confirm your action, or "Cancel" to abort your action.
  6. If you choose "Ok", system should display a message about the action result.
