How To Edit Department

Note: Only System Admin and Department Admin have permissions to edit department information.

  1. Click on the "Departments" button on the left side menu bar
  2. On "Departments" page, you should see a table of all current departments in system
  3. Find the department you want to edit using the "Filter entries" function by inputing relevant department code or name on the "Filter entries" box, or you could click on any sortable table column header to make your search easier.
  4. Press the modify icon on the row you want to edit, this would bring you to the "Modify Department" form containing current department information.
  5. You can input new department code (e.g. ICT) and department name in the form.
  6. Click the "Save" button to save changes, click the "Reset" to populate original department information, or click "Cancel" button to navigate back to "Departments" page.
