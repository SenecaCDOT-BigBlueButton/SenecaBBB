How To Edit Event Schedule

Note: Only future events in a schedule would be updated, this update would not affect past events in your schedule

  1. Navigate to "Edit Event Schedule" page, Option 1:
    • Click on the "Events" button on the left side menu bar.
    • Find lecture or meeting you want to edit using the "Filter entries" function by inputing event title or description on the "Filter entries" box, or you could click on any sortable table column header to make your search easier.
    • Press the details icon on the row you want to edit, this would bring you to the "Current Event" page.
  2. Navigate to "Edit Event Schedule" page, Option 2:
    • Click on the "Calendar" button on the left side menu bar.
    • Find lecture or meeting you want to edit, you can check or uncheck the filter options to help your search
    • Press the event on calendar you want to edit, this would bring you to the "Current Event" page.
  3. In the "Event Schedule" section, click on edit icon , an "Edit Event Schedule" form containing current event information would be displayed
  4. You can modify event title, description.
  5. Select event start date on calendar by click on icon , or select year, month and day from drop down list, this must be current or future date
  6. Click on the "Start Time" box and select start time from drop down list. If the start date is current date, event start time must be later than current time!
  7. Modify in minutes of event duration.
  8. Choose event recurrence from "Only once","Daily","Weekly",and "Monthly"
  9. Recurrence "Only Once": Single event without repeating.
  10. If you choose recurrence by "Daily",please specify number of DAYS you want to repeat the event in the "Repeat Every" box ,
    specify number of events if you choose ends on "After # of occurrences"
    ,
    or specify end date if you choose ends on specified date
  11. If you choose recurrence by "Weekly",please specify number of WEEKS you want to repeat the event in the "Repeat Every" box ,
    specify days of the week for your event.


    specify number of occurrences if you choose ends on "After # of occurrences"
    ,
    specify end date if you choose ends on specified date

    or specify number of occurrences if you choose ends on "After # of Weeks"
  12. If you choose recurrence by "Monthly",please specify number of MONTHS you want to repeat the event in the "Repeat Every" box,
    choose from Occurs by "Day of the month" or "First occurrence of the day of the week"
    specify "Day of the month" from drop down list and input number of occurrences for your event if you choose Occur by "day of the month"
    or choose the day of week and input number of occurrences for your event if you choose occur by "First occurrence of the day of the week"
  13. Click the "Save" button to save changes, click the "Reset" to populate original settings, or click "Cancel" button to navigate to "View Event Schedule" page.
