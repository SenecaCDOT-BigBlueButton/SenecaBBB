How To Edit Lecture

Note: Please edit upcoming lecture only!

  1. Navigate to "Current Event" page, Option 1:
    • Click on the "Events" button on the left side menu bar.
    • Find lecture you want to edit using the "Filter entries" function by inputing event title or description on the "Filter entries" box, or you could click on any sortable table column header to make your search easier.
    • Press the details icon on the row you want to edit, this would bring you to the "Current Event" page.
  2. Navigate to "Current Event" page, Option 2:
    • Click on the "Calendar" button on the left side menu bar.
    • Find lecture you want to edit, you can check or uncheck the filter options to help your search
    • Press the event on calendar you want to edit, this would bring you to the "Current Event" page.
  3. In the "Current Event" section, click on modify icon , an "Lecture Event" form containing current event information would be displayed
  4. You can modify event description.
  5. To modify event start time, click on the "Start Time" box and select start time from drop down list. If the start date is current date, event start time must be later than current time!
  6. Modify minutes of event duration if you want
  7. You can also cancel current event, enable or disable recording
  8. Click the "Save" button to save changes, click the "Reset" to populate original settings, or click "Cancel" button to navigate to "Current Event" page.
