How To Edit Subject

Note: Only System Admin and Department Admin have permissions to edit subjects.

  1. Click on the "Subjects" button on the left side menu bar
  2. On "Subjects" page, you should see a table of all current course sections in system
  3. Click the "Manage Subjects" Button, this would bring you to the "Subject List" page which contains all subjects in system.
  4. Find the subject you want to edit using the "Filter entries" function by inputing relevant subject id or subject name on the "Filter entries" box, or you could click on any sortable table column header to make your search easier.
  5. Click on the edit icon on the row you want to edit.
  6. System should bring you to the "Edit Subject" page contains current subject information.
  7. The "Subject Code" is NOT editable, you could edit the "Subject Name" only!
  8. Click the "Update" button to save changes, click the "Reset" to populate original department information, or click "Cancel" button to navigate back to "Subject List" page.
