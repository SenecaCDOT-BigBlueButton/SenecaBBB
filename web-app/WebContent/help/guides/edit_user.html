How To Edit User

Note: To edit user nick name and user role, ban the user from system, appoint system admin, activate or deactivate user account

  1. Click on the "Manage Users" button on the left side menu bar.
  2. Find the user you want to edit using the "Filter entries" function by inputing username on the "Filter entries" box, or you could click on any sortable table column header to make your search easier.
  3. Press the edit icon on the row you want to edit, this would bring you to the "Edit User Information" page.
  4. You could edit user nick name and user role, ban the user, appoint this user as super admin, activate or deactivate this user.
  5. Click the "Update" button to save changes, click the "Reset" to populate original settings, or click "Cancel" button to navigate back to "Manage Users" page.
