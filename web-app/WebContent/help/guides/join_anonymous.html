How To Join BigBlueButton Conference

Note: After you received an email invitation from BBBMan users, make sure to check the event start time (ISO 8601 format).

  1. To join the meeting, simply click on the URL link which is included in the invitation email.
  2. You should see a welcome page containing brief introduction about BigBlueButton Web Conference System.
  3. Type in your Name in the input box besides the "Join" button.
  4. Click on the "Join" button to join BigBlueButton Web Conference.
  5. If the meeting has not started or has already ended, you should get an alert message.
