How To Manage Meeting Attendee

Note: Event creators could add event attendees (internal users who have valid Seneca username and password, or guest users who have BBBMan account) to join your events.

  1. Navigate to "Current Event" page, Option 1:
    • Click on the "Calendar" button on the left side menu bar.
    • Find the meeting you want to add attendees
    • click on the meeting on calendar, this would bring you to the "Current Event" page.
  2. Navigate to "Current Event" page, Option 2:
    • Click on the "Events" button on the left side menu bar.
    • Find the meeting you want to add attendees using the "Filter entries" function by inputing event title or description on the "Filter entries" box, or you could click on any sortable table column header to make your search easier.
    • Click the details icon on the row you want to manage, this would bring you to the "Current Event" page.
  3. Click on the "Manage Attendee" button located on the top section of page, this would bring you to "Add Meeting Attendee" page
  4. To add Seneca internal users, please input valid Seneca student or employee full username into the top first "Add User" input box.
  5. Press "Enter" key or click the search icon to add user to attendee list
  6. A "Success", "User Not Found", or "User already added" information would show on top of the page, if success, you should be be able to find the user on the meeting attendee list table.
  7. To add Guest users, please input full or partial username into the second "Search User" input box to search any matched users in system.
  8. If system found any matched guest users, it would display a table of search result. You could click on the green color "Add" icon to add the guest to attendee list
  9. An email notification, including meeting creator name and meeting start time, would be sent to each of your meeting attendee.
