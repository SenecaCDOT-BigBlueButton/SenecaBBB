How To Set/Remove Department Administrator Role

Note: Only System Admin and Department Admin have permissions to change departments users' admin status.

  1. Click on the "Departments" button on the left side menu bar
  2. On "Departments" page, you should see a table of all current departments in system
  3. Find the department you want to modify using the "Filter entries" function by inputing relevant department code or name on the "Filter entries" box, or you could click on any sortable table column header to make your search easier.
  4. Click the users icon on the row you want to modify, this would bring you to the "Department Users" page which contains all department users in current department.
  5. To add admin role to the user, click on the Set Admin Status icon on the row you want to modify.
  6. The user "Admin Status" column should change to "YES".

  7. To remove admin role to the user, click on the Set Admin Status icon on the row you want to modify.
  8. The user "Admin Status" column should change to empty.
