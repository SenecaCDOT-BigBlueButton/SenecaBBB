How To Start Meeting/Lecture

Note: You can start the event 15 minutes before scheduled time. If the event is expired, please create a new one.

  1. Navigate to "Current Event" page, Option 1:
    • Click on the "Calendar" button on the left side menu bar.
    • Find the event you want to start in current date
    • Press the event on calendar you want to start, this would bring you to the "Current Event" page.
  2. Navigate to "Current Event" page, Option 2:
    • Click on the "Events" button on the left side menu bar.
    • Find the event you want to start using the "Filter entries" function by inputing event title or description on the "Filter entries" box, or you could click on any sortable table column header to make your search easier.
    • Press the details icon on the row you want to start, this would bring you to the "Current Event" page.
  3. In the "Current Event" section, if the "Start Meeting" or "Start Lecture" button's background color is red, you could click on it, and this would bring you to BigBlueButton Web Conference System.
  4. If the "Start Meeting" or "Start Lecture" button's background color is grey, that means it's too early or too late to start your event.
