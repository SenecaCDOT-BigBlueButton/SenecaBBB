How To Terminate Meeting/Lecture

Note: You can terminate an active event any time.

  1. Navigate to "Current Event" page, Option 1:
    • Click on the "Calendar" button on the left side menu bar.
    • Find the event you want to terminate
    • Click the event on calendar you want to terminate, this would bring you to the "Current Event" page.
  2. Navigate to "Current Event" page, Option 2:
    • Click on the "Events" button on the left side menu bar.
    • Find the event you want to terminate using the "Filter entries" function by inputing event title or description on the "Filter entries" box, or you could click on any sortable table column header to make your search easier.
    • Click on the details icon on the row you want to terminate, this would bring you to the "Current Event" page.
  3. In the "Current Event" section, click on the "Terminate Event" Button to terminate current event.
