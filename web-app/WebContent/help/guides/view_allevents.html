How To View All Events

Note: System admins have permissions to view all events (including all users' events) in BBBMan database.

  1. Click on the "Events" button on the left side menu bar, system would list all meetings and lectures for you.
  2. To find lecture or meeting you want to view using the "Filter entries" function by inputing event title or description on the "Filter entries" box, or you could click on any sortable table column header to make your search easier.
  3. To find a single event details, click on the details icon on the row you want to view, this would bring you to the "Current Event" details page.
