How To View Event Schedule

Note: Event Schedule information includes A list of events and a list of attendees in current schedule

  1. Navigate to "View Event Schedule" page, Option 1:
    • Click on the "Events" button on the left side menu bar.
    • Find lecture or meeting you want to view using the "Filter entries" function by inputing event title or description on the "Filter entries" box, or you could click on any sortable table column header to make your search easier.
    • Press the details icon on the row you want to edit, this would bring you to the "Current Event" page.
  2. Navigate to "View Event Schedule" page, Option 2:
    • Click on the "Calendar" button on the left side menu bar.
    • Find lecture or meeting you want to view, you can check or uncheck the filter options to help your search
    • Press the event on calendar you want to view, this would bring you to the "Current Event" page.
  3. In the "Event Schedule" section, click on detail icon , an "View Event Schedule" page containing a list of events would be displayed
